## 📌 프로젝트 개요

외부 라이브러리 없이 순수 HTML, CSS, JavaScript만으로 구현한 웹 게임 프로젝트입니다.  
브라우저에서 바로 실행되며, 기본 웹 기술에 대한 이해를 바탕으로 개발했습니다.

---

## 🚀 실행 방법

```bash
# 별도 설치 없이 바로 실행 가능
1. 레포지토리 클론
git clone https://github.com/Limdangun/dangun-week1.git

2. 폴더 이동
cd dangun-week1

3. soccer_game.html 파일을 브라우저로 열기
```
